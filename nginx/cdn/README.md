# `gecut/nginx/cdn` v2

CDN-origin specialization built on `gecut/nginx/core:2.0.0`. It serves static
files directly; it is not a reverse proxy or an on-disk proxy cache.

## Cache policy

`Cache-Control` is generated by one URI/status policy map. The inherited
`expires` policy is disabled, so responses do not contain competing cache
directives.

| Content | Default policy |
| --- | --- |
| Webpack/CRA/Vite fingerprinted asset | `public, max-age=31536000, immutable, s-maxage=31536000` |
| Unversioned static file | `public, max-age=3600, stale-while-revalidate=30` |
| HTML or directory shell | `public, max-age=0, must-revalidate, s-maxage=0` |
| JSON or web manifest | `public, max-age=60, stale-while-revalidate=30` |
| `service-worker.js` or `sw.js` | `no-cache` |
| Any `4xx` or `5xx` response | `no-store` |
| Any method other than GET/HEAD | `no-store` |

ETag is enabled. `Vary: Accept-Encoding` is always emitted; JPEG/PNG resources
also vary on `Accept` for optional WebP negotiation.

## Environment

| Variable | Default |
| --- | --- |
| `NGINX_CDN_CACHE_HASHED` | `public, max-age=31536000, immutable` |
| `NGINX_CDN_S_MAXAGE` | `31536000` |
| `NGINX_CDN_CACHE_UNVERSIONED_STATIC` | `public, max-age=3600, stale-while-revalidate=30` |
| `NGINX_CDN_CACHE_HTML` | `public, max-age=0, must-revalidate` |
| `NGINX_CDN_HTML_S_MAXAGE` | `0` |
| `NGINX_CDN_CACHE_JSON` | `public, max-age=60, stale-while-revalidate=30` |
| `NGINX_CDN_CACHE_SERVICE_WORKER` | `no-cache` |
| `NGINX_CDN_CACHE_ERROR` | `no-store` |

Values must not contain control characters or quotes.

## Run

```bash
docker run --rm -p 8080:80 -v "$PWD/public:/data:ro" \
  ghcr.io/gecut/nginx/cdn:2.0.0
```

```bash
curl -sI http://localhost:8080/app.abc12345.js | \
  grep -Ei 'cache-control|etag|vary'
```

## Migration from v1

- `expires max` has been removed; `Cache-Control` is the only cache authority.
- Vite-style hyphenated hashes and service-worker policies are recognized.
- Errors are never cached.
